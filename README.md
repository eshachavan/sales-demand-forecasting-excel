# 📊 Sales Demand Forecasting using Excel

## 📌 Project Overview
This project focuses on analyzing 12 months of historical sales data to identify trends and forecast future demand using Microsoft Excel. The goal is to understand sales patterns and predict future performance for better business decision-making.

---

## 🧰 Tools Used
- Microsoft Excel  
- Line Chart 
- FORECAST.LINEAR function 

---

## 📈 Methodology
1. Collected and structured monthly sales data.
2. Analyzed trends using line charts.
3. Applied `FORECAST.LINEAR` function for prediction. 
4. Visualized actual vs forecasted sales. 

---

## 🔮 Forecasting Technique
Used **Linear Regression (FORECAST.LINEAR)** in Excel to predict future sales based on historical trends.
